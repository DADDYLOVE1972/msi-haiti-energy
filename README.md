# MSI Haiti Energy

Professional React + Vite + Tailwind CSS website for MSI Haiti Energy.

## Pages

- Home
- About
- Marigot Project
- Impact
- Partner With Us
- Donate
- Contact

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The Donate page is intentionally inactive and marked as coming soon. The Contact form is frontend-ready and can be connected to email, CRM, or donation platform workflows.
